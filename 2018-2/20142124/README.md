+ # 창의코딩스마트앱
  + ## 기말 프로젝트
    + ### 자가 심리-돌봄 도우미 앱(Self Psycho-Care Helper App)
    
        + #### 소개
        
        개인의 정신건강을 점검하고, 증진시킬 수 있도록 돕는 앱입니다.
        ***
        
        + #### 주요 기능
        
        심리검사, 사고기록지, 날짜별 검사결과 그래프
        ***
        
        + #### 주요 컴포넌트
        
        Firebase DB, Tiny DB, canvas
